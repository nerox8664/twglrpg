# twglrpg
### About
This is free realization of tiled RPG game used webgl.
### Installation
1. Install Node.js version 0.12+
2. Install global gulp
`npm install -g gulp`
3. Clone this repository
`git clone https://github.com/nerox8664/twglrpg`
4. Install packages
```
$ cd twglrpg
$ npm install
$ bower install
```
5. Get fun with `gulp`
