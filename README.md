# TWGLRPG
This is free realization of tiled RPG game using WEBGL (Phaser.js).

### Installation
1. Install latest version Node.js
2. Install global Gulp
`npm install -g gulp`
3. Clone this repository
`git clone https://github.com/nerox8664/twglrpg`
4. Install packages

```
$ cd twglrpg
$ npm install
$ bower install
```
### Usage

Get fun with `$ gulp`.
